# 启动说明

## 后端服务器
后端服务器已在后台运行，监听端口 3001。

## 前端服务器 - 权限问题修复

由于 `node_modules` 目录的所有者是 root，需要先修复权限才能启动前端服务器。

### 方法 1：修复权限（推荐）
在终端中运行以下命令：
```bash
cd /Users/et/Desktop/Learning/frontend
sudo chown -R $(whoami) node_modules/.vite-temp
```

然后启动前端：
```bash
npm run dev
```

### 方法 2：使用启动脚本
```bash
cd /Users/et/Desktop/Learning/frontend
./start-frontend.sh
```

### 方法 3：手动创建符号链接
```bash
cd /Users/et/Desktop/Learning/frontend
mkdir -p /tmp/vite-temp-$(whoami)
sudo rm -rf node_modules/.vite-temp
sudo ln -s /tmp/vite-temp-$(whoami) node_modules/.vite-temp
npm run dev
```

## 服务地址
- 后端 API: http://localhost:3001
- 前端 UI: http://localhost:5173 (启动后查看实际端口)
